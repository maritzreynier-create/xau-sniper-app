# XAUUSD Sniper Dashboard — Vercel Package

## Deploy on Vercel
1. Go to Vercel and sign in.
2. Click **Add New...** then **Project**.
3. Choose **Import / Upload** and upload this folder or the zip.
4. Deploy.
5. Open your live URL on your phone.
6. In Chrome or Safari, use **Add to Home Screen**.

## Files
- `index.html` — the app
- `vercel.json` — routing config

## Notes
- Data is stored in the browser's local storage.
- Use the in-app **Export JSON Backup** button regularly.
- Local storage data is per browser/device unless you later add cloud sync.
